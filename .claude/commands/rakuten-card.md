楽天アフィリエイトのHTMLリンクから、MDX用の `RakutenCardImage` と `RakutenCard` コンポーネントを生成してください。

## 入力

`$ARGUMENTS` に楽天アフィリエイトのHTMLが含まれている場合はそれを使用してください。
引数が空の場合は、ユーザーに楽天アフィリエイトのHTMLを貼り付けるよう求めてください。

## 手順

1. HTMLから以下の情報を抽出する
   - **href**: 商品ページへのアフィリエイトリンクURL（`<a href="...">` の最初のもの）
   - **imgSrc**: 商品画像URL（`<img src="...">` の最初のもの、楽天ロゴ画像 `rl.svg` は除く）
   - **title**: 商品タイトル（`<a href="...">` のテキストコンテンツ）
   - **price**: 価格テキスト（「価格：〇〇円（税込...）」の部分。「円」以降の補足は含めず `〇〇円（税込、送料無料）` 形式にする）

2. 以下のMDXコンポーネントを生成する

```mdx
<RakutenCardImage
  href="【href】"
  imgSrc="【imgSrc】"
  title="【商品タイトルの短縮版（製品名のみ、30文字程度）】"
/>

<RakutenCard
  href="【href】"
  imgSrc="【imgSrc】"
  title="【商品タイトルのフルテキスト】"
  price="【price】"
/>
```

3. 生成したコードをユーザーに提示する

4. 挿入先のMDXファイルを確認する
   - IDEで開いているファイルがある場合はそれを候補として提示する
   - ファイルが特定できない場合はユーザーに確認する

5. 承認されたら以下の位置にコンポーネントを挿入する
   - **`RakutenCardImage`**: フロントマター（`---` で囲まれたブロック）直後の本文先頭
   - **`RakutenCard`**: 記事の末尾（最後の行の**直後**、ファイル末尾）

## コンポーネントの仕様

- `RakutenCardImage`: 商品画像のみのリンク表示。フロントマター直後の本文先頭に挿入する
- `RakutenCard`: 画像・タイトル・価格・楽天購入ボタンを含むカード。「ではまた」などの締めの一文の**直後**（ファイル末尾）に挿入する
- 両コンポーネントとも `src/components/mdx/RakutenCard.tsx` で定義済み

## 注意事項

- `title` の短縮版は製品名・シリーズ名が伝わる範囲で簡潔にする
- URLはそのまま使用し、エンコードを変更しない
- 価格は抽出時点の値であることをユーザーに伝える
